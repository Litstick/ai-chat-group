# AI 聊天群

一个支持多 AI 角色协作讨论的群聊应用。群内仅有一个真实人类用户，其余均为 AI 角色，围绕用户设定的核心话题展开讨论。

## 功能特性

### 核心聊天
- **话题驱动**：开始前设定核心话题（如开发软件、调研报告、数据分析等）
- **AI 自主交流**：多个 AI 角色根据话题自行理解和反馈，模拟真实群聊氛围
- **人类随时参与**：用户可随时发送消息加入讨论，AI 会针对用户发言进行回应
- **消息持久化**：所有聊天记录自动保存到浏览器 localStorage

### AI 配置
- **角色选择**：至少选择 2 个、最多 5 个 AI 参与群聊
- **角色描述**：为每个 AI 设置个性化角色和专长描述
- **Skill 配置**：为 AI 配置代码审查、数据分析、文档编写、头脑风暴、翻译、搜索等技能

### 设置管理
- **群活跃时间段**：设置 AI 参与群聊的时间段
- **AI 视听能力**：单独开关图片理解、表情包回复、语音回复、视频分享
- **聊天时长限制**：设定自动终止时间
- **模型管理**：添加、删除、设置默认 AI 模型

### 会话管理
- **一键总结**：生成重要信息和次要信息的分类总结
- **终止聊天**：用户可随时结束当前会话
- **历史记录**：查看过往所有会话，支持重新进入查看

## 技术栈

- **前端框架**：React 18 + TypeScript
- **状态管理**：Zustand
- **样式方案**：Tailwind CSS
- **构建工具**：Vite
- **图标库**：Lucide React

## 快速开始

### 环境要求
- Node.js >= 18
- npm >= 9

### 安装依赖

```bash
cd ai-chat-group
npm install
```

### 开发模式

```bash
npm run dev
```

应用将在 `http://localhost:5173` 启动。

### 生产构建

```bash
npm run build
```

构建产物输出到 `dist/` 目录。

## 使用说明

1. **首次使用**：进入「AI 配置」页面，选择至少 2 个 AI 角色参与群聊，可为其设置角色描述和 Skill
2. **新建聊天**：在首页点击「新建聊天」，输入核心话题后开始讨论
3. **参与讨论**：在聊天界面输入消息，AI 们会围绕话题自主交流，也会回应你的发言
4. **一键总结**：点击「一键总结」按钮，快速查看聊天中的重要信息
5. **终止聊天**：点击「终止聊天」按钮结束当前会话，历史记录会自动保存
6. **查看历史**：在「历史记录」页面查看过往会话，点击可重新进入查看完整记录

## 数据存储

所有数据均存储在浏览器的 localStorage 中，包括：
- 聊天会话记录
- AI 角色配置
- 应用设置
- Skill 配置

清除浏览器数据将导致所有记录丢失。
